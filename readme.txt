Use your E-Goi account within Wordpress. Send Emails, capture contacts and configure your subscribe form.

== Description ==

This plugin allows you to use some of the API functions provided by E-Goi (http://e-goi.com/s/e-goi/40301803c0) inside WordPress. With this plugin you can: 
* Create Email Marketing Campaign using wordpress editor. 
* Send Email campaings to your mailinglists. 
* Create automatic newsletters with the latests posts published in your Blog (Based on the numbe of posts). 
* Place a widget in your blog with a fully configurable subscribption form. 
* Sync registered wordpress users to a E-Goi List.

NOTE: Don't have a E-Goi account, register one for FREE (http://e-goi.com/s/e-goi/40301803c0) With a free account you can send unlimited Email Marketing Campaigns up to 500 subscribers.

== Installation ==

1. Upload `s9-egoi` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Egoi options page, insert your E-Goi API Key and configure the plugin behaviour.
4. Place the Egoi Widget in your sidebar if you want the Webform for subscriptions.

== Frequently Asked Questions ==

= What is E-Goi? =
E-Goi is a Multichannel Marketing Platform that allows you to keep multiple mailinglists. Send Marketing campaigns via Email, SMS, MMS, Voice broadcasting and Fax - Track all statistics related to every campaign sent. 
If you don't have a E-Goi account, register one for FREE (http://e-goi.com/s/e-goi/40301803c0) With a free account you can send unlimited Email Marketing Campaigns up to 500 subscribers.

= Where do i find my API Key? =
After you login in your E-Goi account, create a subscriber list and click on the options link of that list. You will see a API button on the left and after u click it you should find your API key.
